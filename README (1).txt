OPCIÓN A — Usar con GitHub Pages (GRATIS, recomendado)
────────────────────────────────────────────────────────
1. Crea cuenta en https://github.com
2. Crea un repositorio nuevo (ej: "mis-finanzas")
3. Sube los 3 archivos: index.html, manifest.json, icon.png
4. Ve a Settings → Pages → Branch: main → Save
5. Tu app queda en: https://TU-USUARIO.github.io/mis-finanzas

6. En el iPhone, abre Safari y entra a esa URL
7. Toca el ícono de compartir (cuadrado con flecha ↑)
8. Selecciona "Añadir a pantalla de inicio"
9. ¡Listo! Aparece como app en tu pantalla


OPCIÓN B — Netlify Drop (la más rápida, sin cuenta)
────────────────────────────────────────────────────
1. Ve a https://app.netlify.com/drop en tu computador
2. Arrastra la carpeta completa con los 3 archivos
3. Obtienes una URL del tipo: https://nombre-random.netlify.app
4. Abre esa URL en Safari del iPhone
5. Toca compartir → "Añadir a pantalla de inicio"


NOTAS IMPORTANTES
─────────────────
• Los datos se guardan en el iPhone (localStorage de Safari)
  No se borran al cerrar, solo si borras datos de Safari
• Para respaldar: usa la opción exportar (puedes agregar esa
  función más adelante)
• No necesita internet una vez instalada (funciona offline)
• Compatible con iPhone con iOS 11.3 o superior


ARCHIVOS INCLUIDOS
──────────────────
  index.html    → La app completa
  manifest.json → Configuración para iOS
  README.txt    → Este archivo

Crea un icono (icon.png de 512x512px) para que se vea
mejor en la pantalla de inicio. Puedes usar cualquier
imagen y renombrarla "icon.png".
